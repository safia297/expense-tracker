# 💰 Expense Tracker

A full-stack expense tracking app built with React and Node.js.

## Features
- Add income and expenses by category
- Real-time balance, income and expense summary
- Doughnut chart showing spending by category
- Delete transactions

## Tech Stack
**Frontend:** React, Chart.js, Vite  
**Backend:** Node.js, Express, SQLite

## How to Run

### Backend
```bash
cd backend
npm install
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
